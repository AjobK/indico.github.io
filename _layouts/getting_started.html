Getting started with Indico

Indico has a few easy ways to quickly get started, each one appealing to a different skill level and use case. Read through to see what suits your particular needs.
INSTALL INDICO ON YOUR OWN MACHINE
DOWNLOAD A CLOUD IMAGE
TRY IT OUT (SANDBOX)

Installing Indico

Indico can be installed using easy_install:

easy_install indico

You can find both source packages and Python eggs on Github.

You will need to perform a couple of additional configuration tasks, which are explained in our docs.

Indico Cloud Image

If you want to deply Indico on the cloud, there are several options:

  1. Choose and download an existing Indico Cloud Image
  2. Generate your own Indico Cloud Image, using the image creation script. Instructions to walk you through the process can be found in our guide.
  3. Configure an Indico Cloud Image (on Amazon, Rackspace, etc.), following the instructions.
